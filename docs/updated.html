Snap Links

Snap Links is a Browser Addon that lets you lasso links, checkboxes and other elements and do things with them.

New Version Installed

Your version of Snap Links has just been updated, here is what is new:

v3.1.1

  • A few configuration options now available
    • Enable/Disable switching focus to new tab.
    • Show/Hide the number of selected links.
